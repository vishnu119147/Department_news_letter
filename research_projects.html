Research Projects

Here are the latest research projects done by BIET College.

Safety Gadget: An IoT Based Security System

The Safety Gadget IoT-based security system represents a significant advancement in the field of security technology. By leveraging the power of IoT, it provides a flexible, scalable, and efficient solution for monitoring and protecting various environments.

IoT Based Animal Tracking System

An IoT-based animal tracking system uses interconnected sensors and GPS technology to monitor the location and behavior of animals in real-time. The system typically involves lightweight GPS collars or tags attached to the animals, which transmit data to a central cloud server via IoT gateways.

Detection Of Cyberbullying On Social Media Platforms

Cyberbullying on social media platforms has become a significant concern due to its widespread impact on mental health and well-being. Detecting and mitigating cyberbullying requires advanced technologies and methodologies to analyze vast amounts of data in real-time.

Depression Detection System and Remedial Recommendation System Using Machine Learning

This system uses machine learning algorithms to detect signs of depression from text, speech, and behavioral data. It provides personalized remedial recommendations, offering support and interventions based on the user's data.

Smart Ward Tracking and Alerting System

The Smart Ward Tracking and Alerting System utilizes IoT and sensors to monitor patients in real-time, providing alerts to medical staff for immediate attention. This system aims to enhance patient care and safety in hospital wards.

Novel Approach to Formulate Zone Parameters to Improve the Crop Yield Using Remote Sensing Data for Precision Agriculture

This research focuses on using remote sensing data to create zone parameters for precision agriculture. The approach aims to improve crop yield by providing farmers with precise information on soil conditions, moisture levels, and crop health.
